## 📋 Содержание

| Файл | Задача | Описание |
|------|--------|----------|
| `test1.java` | Арифметические операции | Программа выполняет сложение, вычитание, умножение и деление двух чисел |
| `test2.java` | Сравнение строк | Программа сравнивает две строки и выводит "Строки идентичны" или "Строки неидентичны" |

3. Нажми зелёную стрелку ▶ рядом с методом `main`
